UID Documentation
=================

UID is an all-in-one ID toolkit for PHP with generation, parsing, validation, and conversion APIs.

It supports:

- UUID (v1, v3, v4, v5, v6, v7, v8)
- ULID
- Snowflake
- Sonyflake
- Randflake
- TBSL
- NanoID and CUID2
- KSUID and XID
- Opaque and deterministic IDs
